⚠ Model Agreement Note

The 81% model agreement rate reflects agreement between the logistic regression model and the deterministic rule engine on a held-out test set of 100 patients (80/20 train/test split). The model learned the rule logic and can reproduce it consistently. This does not validate clinical outcomes. Full clinical validation requires real patient follow-up data and is planned for Phase 2. Planned upgrade: HITL validation by a licensed PT on 50–75 edge cases.

ℹ Equity Monitoring Rationale

Demographic variables are included for post-hoc monitoring only. Sex, race/ethnicity, occupation, and region are not features in the rule engine or the logistic regression model. Pathway assignment is determined solely by VAS pain score, pain duration, and Oswestry Disability Index score. Two patients with identical clinical inputs receive the same pathway regardless of demographic profile. Any statistically significant disparity across groups should be reviewed as a potential proxy-variable bias signal.
